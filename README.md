# HRD Corp Eligibility Calculator

A standalone React app that estimates training grant eligibility based on the **HRD Corp Allowable Cost Matrix (ACM)** — Malaysia.

> ⚠️ Independent estimation tool. Not affiliated with or endorsed by HRD Corp.

## Features

- Schemes: HCC, SBL, SLB
- Training types: In-House, ROT, Public, E-Learning, Overseas, Development Programme
- Development Programme levels: PhD, Masters, Degree, Diploma, SKM (Level 1–5)
- Actual course fee vs ACM ceiling comparison with deficit calculation
- Notes & eligibility breakdown per component

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- React 18
- Pure JS calculation engine (no backend required)
